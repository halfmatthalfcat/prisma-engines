Expand description

A type level number signifying the number of bits in a bitmap.

This trait is implemented for type level numbers from U1 to U128.

Examples

assert_eq!(
    std::mem::size_of::<<U10 as Bits>::Store>(),
    std::mem::size_of::<u16>()
);
